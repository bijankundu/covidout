# Description

This is a covid - 19 tracker web app. It pulls data from various APIs and updates the data in realtime.

## Technologies used

- HTML
- CSS
- NodeJs
- ExpressJs

### Dependencies

- body-parser
- EJS
- lodash
- express
- node-fetch
- unirest

### APIs Used

1.  [Novelcovid api ](https://github.com/novelcovid/api)
2.  [Covid19india api](https://github.com/covid19india/api)
